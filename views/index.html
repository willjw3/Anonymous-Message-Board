Features

  1. Prevents clickjacking by allowing only your site to be loaded in an iFrame on your own pages.
  2. Does not allow DNS prefetching.
  3. Allows only your site to send the referrer for your own pages.
  4. Users can POST a new thread to a specific message board.
  5. Users can POST a reply (comment) to a thead on a specific board.
  6. Users can GET the most 10 most recently bumped threads on the board with only the 3 most recent replies.
  7. Users can GET an entire thread with all it's replies.
  8. Users can delete a thread completely.
  9. Users can delete a reply (comment).
  10. Users can report a thread.
  11. Users can report a reply (comment).
  12. Contains a suite of functional tests.
